# Getting Started

GET "http://localhost:8081/api/persons/"   : Return All Records

GET "http://localhost:8081/api/persons/{id}"   : Return given Records

POST "http://localhost:8081/api/persons/" : Add new record.

Example of request
{
"name": "JBT",
"city": "India",
"height": 4,
"dob": "2021-01-01"
}
